# First Real User Observation Feedback

## Session Guide (Observer Instructions)
- **Remain silent**: Let the user navigate alone.
- **Do not guide**: Only step in if the user is completely blocked/unable to proceed.
- **Capture hesitations**: Note the first point where the user pauses or looks confused.
- **Observe trust signals**: Look for signs of doubt or questioning of AI outputs/data.
- **Record misunderstandings**: Document what the user expected vs. what actually happened.
