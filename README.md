# Payd.ai
AI-powered accounts receivable collection engine for agencies.

## Core Pillars
- **Promise Tracker:** Auto-extracting payment dates from email replies.
- **CRS:** Compounding Client Reliability Scores.
- **Action Queue:** Human-in-the-loop email drafting.
- **Shadow Mode:** Mandatory manual approval during onboarding.

## Technical Summary
- **Stack:** Next.js, Supabase, Anthropic Claude, Resend.
- **Primary Sending:** OAuth "Send-as-User".
